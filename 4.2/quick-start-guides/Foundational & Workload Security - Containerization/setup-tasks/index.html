    ```shell script all Runs all setup tasks download-ca-cert Download CMS root CA certificate download-cert-tls Download CA certificate from CMS for tls database Setup authservice database admin Add authservice admin username and password to database and assign respective roles to the user jwt Create jwt signing key and jwt certificate signed by CMS create-credentials Generates credentials to support third party authentication and authorization update-service-config Sets or Updates the Service configuration
